# Analise-de-Desempenho-de-Equipe-Python-POO-
Este projeto foi desenvolvido com o objetivo de praticar Programação Orientada a Objetos (POO) em Python, simulando a análise de desempenho de uma equipe comercial.
A aplicação permite cadastrar membros da equipe e calcular métricas importantes de desempenho, organizando os dados de forma estruturada.

- Funcionalidades
Cadastro de membros com nome e rendimento, 
Cálculo do rendimento total da equipe, 
Cálculo da média de vendas, 
Identificação do maior e menor desempenho, 
Geração automática de ranking em ordem decrescente.

- Conceitos aplicados
Classes e métodos, 
Encapsulamento básico, 
Estruturas de dados (listas e dicionários), 
Funções lambda, 
sorted(), max() e min(), 
enumerate() para geração de ranking.

- Objetivo do projeto

Aprimorar a base em lógica de programação, organização de código e modelagem de dados utilizando POO, simulando um cenário próximo à análise de desempenho real.
Projeto desenvolvido para fins de estudo.
